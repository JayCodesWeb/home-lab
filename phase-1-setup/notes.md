# Phase 1 Setup Notes

## What I did
I installed VirtualBox and created Windows 10 virtual machine

## Setup Details
- Host OS: Windows 11 (64-bit)
- VirtualBox version: 7.2.8
- VM specs: 4GB RAM, 50GB virtual disk
- OS installed: Windows 10 Version 22H2 (64-bit)

## Process
I downloaded VirtualBox from virtualbox.org and the Windows 10 ISO from Microsoft. I created a new VM, mounted the ISO, and completed the Windows installation. I took an initial snapshot named "Clean Install" as a restore point.

## What I learned
How to configure VM hardware settings and understand the difference between a host machine and a guest machine.
